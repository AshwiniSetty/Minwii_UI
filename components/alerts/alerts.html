Alerts
An alert is a message that is displayed in a way that attracts the user's attention without interrupting their flow.
Simple Alerts
To generate a simple alert use the base class alert. There are 5 types of themed alerts - Primary, Secondary, Success, Warning, and Danger alerts.

To generate a themed simple alert, use any of the following theme class along with the base class- alert-primary, alert-secondary, alert-success, alert-warning, and alert-danger.
