Buttons
Buttons have an important essence to any interactive Web application. Buttons are components of a website, that a user can click to execute an action, make choice or interact with the website.
Contained Button
Contained buttons are high-emphasis, distinguished by their use of elevation and fill. They contain actions that are primary to your app. These can be considered for those things that we want the user to do as often as possible and should be as visible as possible.
Use the default 'btn' class and the required button variant.
Note: By just using the 'btn' class, default type button is generated.

Disabled Buttons
A disabled button is unusable and un-clickable. The disabled attribute can be set to keep a user from clicking on the button until some other condition has been met.
The class 'btn-disabled' with disabled attribute makes a button unclickable.
NOTE: You can use the any type of button and make it disabled.
Icon Buttons
An icon button is an icon that triggers some sort of action on the page. More accurately, technically speaking, an icon button is a button that contains an icon and no (visible) accompanying text.
The class 'btn-icon' can be used for icon buttons.
NOTE: You can use the any type of button and make it disabled.
Text-Icon Buttons
An icon button is an button that triggers some sort of action on the page that contains an icon and an visible accompanying text.
The required type button class with required text and icon can be used for text-icon buttons.
