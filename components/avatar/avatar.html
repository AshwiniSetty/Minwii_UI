Avatar
Avatar displays user image or initials as profile pictures.
You can create variety of avatars, including Circle, Square and Text Avatars.
These can be utilized in five sizes: extra small, small, medium, large, and extra large.
Circle Avatars
To generate a circle avatar use the base class avatar.
